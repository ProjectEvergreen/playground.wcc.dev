# playground.wcc.dev

Playground repository for [wcc.dev](https://www.wcc.dev) built using [**Greenwood**](https://greenwoodjs.dev/).

## Setup

1. Clone the repository
1. Have NodeJS LTS installed (or run `nvm use`)
1. Run `npm ci`

## Commands

To start the development server, run:

```sh
$ npm run dev
```

To generated a production build, run:

```sh
$ npm run build
```

To start a production server, run:

```sh
# make sure you've generated a production build first

$ npm run serve
```
